# Store My Goods - Feedback Form

A mobile-friendly feedback form built with Next.js 14 and Tailwind CSS.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- ✅ Fully responsive and mobile-friendly
- ✅ Built with Next.js 14 (App Router)
- ✅ Styled with Tailwind CSS
- ✅ TypeScript support
- ✅ Form validation
- ✅ Character counters
- ✅ Conditional fields
- ✅ Clean and modern UI

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- App Router
